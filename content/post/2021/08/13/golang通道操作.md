| 操作  | 通道状态   | 结果                                                         |
| ----- | ---------- | ------------------------------------------------------------ |
| Read  | nil        | 阻塞                                                         |
|       | 打开，非空 | 正常                                                         |
|       | 打开，空   | 阻塞                                                         |
|       | 关闭       | 正常返回值 有值返回 值,true    没有值  返回  通道类型默认值,false |
|       | 只写       | 编译错误                                                     |
| Write | nil        | 阻塞                                                         |
|       | 打开，已满 | 阻塞                                                         |
|       | 打开，未满 | 正常                                                         |
|       | 关闭       | panic                                                        |
|       | 只读       | 编译错误                                                     |
| Close | nil        | panic                                                        |
|       | 打开，非空 | 正常                                                         |
|       | 打开，空   | 正常                                                         |
|       | 关闭       | panic                                                        |
|       | 只读       | 编译错误                                                     |
